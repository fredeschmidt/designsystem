# Input Chip (M3)

Overview
Input chips represent discrete pieces of information entered by a user, such as Gmail contacts or filter options within a search field.

Figma
https://www.figma.com/design/5SfEgZPDQh1YLX96cTyln2/Material-3-Design-Kit--Community---Copy-?node-id=53923-27888&m=dev

Variants
Label only
Label with leading icon
Label with avatar

States
Enabled
Hovered
Focused
Dragged

Selection
Selected and unselected variants are supported for all states.

Trailing icon
A trailing close icon is shown when `data-close="true"` is set.

Accessibility
Buttons expose descriptive `aria-label` text.
Focus indicator is visible for focused states.

Theme tokens
All visual styles are driven by `--ds-` tokens and theme overrides.
