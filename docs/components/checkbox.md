# Checkbox Component

## What is it?
A checkbox component for the design system, supporting checked, indeterminate, and unchecked states, with and without background circles.

## When do we use it?
Use this component wherever a binary or tri-state input is needed in forms or UI.

## Props/API
- States: checked, indeterminate, unchecked
- Visual: with or without background circle

## Examples
See demo in index.html under "Checkbox Component" section.

## Known limitations
- Only supports visual states shown in the screenshot
- No custom icons
